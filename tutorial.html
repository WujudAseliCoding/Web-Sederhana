Tutorial Membuat Website Sederhana

Panduan Step-by-Step dari Nol sampai Online

Persiapan Awal

Tools yang Dibutuhkan:

Text Editor

VS Code, Sublime Text, atau Notepad++

Web Browser

Chrome, Firefox, atau Edge

File Manager

Windows Explorer atau File Explorer

Step 1: Buat Struktur Folder

Buat folder untuk project website Anda dengan struktur seperti ini:

my-website/
├── index.html
├── style.css
├── script.js
└── images/
    └── foto-profil.jpg

Penjelasan:

  • index.html - Halaman utama website
  • style.css - File styling/tampilan
  • script.js - File JavaScript untuk interaktivitas
  • images/ - Folder untuk menyimpan gambar

Step 2: Buat File HTML Dasar

Buat file index.html dan masukkan kode berikut:

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Saya</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header>
    <h1>Selamat Datang di Website Saya</h1>
    <p>Ini adalah website pertama saya!</p>
  </header>
  <main>
    <h2>Tentang Saya</h2>
    <p>Halo! Nama saya [Nama Anda]. Saya sedang belajar membuat website.</p>
  </main>
</body>
</html>

Step 3: Tambahkan CSS Styling

Buat file style.css untuk styling:

/* Reset CSS */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* Body styling */
body {
  font-family: Arial, sans-serif;
  line-height: 1.6;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  min-height: 100vh;
}

/* Header styling */
header {
  background: white;
  padding: 40px;
  text-align: center;
  margin: 20px;
  border-radius: 15px;
  box-shadow: 0 10px 25px rgba(0,0,0,0.1);
}

/* Main content */
main {
  background: white;
  padding: 30px;
  margin: 20px;
  border-radius: 15px;
  box-shadow: 0 10px 25px rgba(0,0,0,0.1);
}

Step 4: Tambahkan JavaScript

Buat file script.js untuk interaktivitas:

// Fungsi untuk menampilkan pesan
function showWelcome() {
  alert('Selamat datang di website saya!');
}

// Jalankan saat halaman dimuat
document.addEventListener('DOMContentLoaded', function() {
  console.log('Website berhasil dimuat!');
  
  // Tambahkan tombol interaktif
  const header = document.querySelector('header');
  const button = document.createElement('button');
  button.textContent = 'Klik Saya!';
  button.onclick = showWelcome;
  header.appendChild(button);
});

Jangan lupa tambahkan link ke JavaScript di HTML:

<script src="script.js"></script>

Step 5: Test dan Deploy

Testing Lokal:

  1. Buka file index.html di browser
  2. Periksa apakah semua styling dan JavaScript berfungsi
  3. Test di berbagai browser (Chrome, Firefox, Edge)
  4. Test responsive di mobile (F12 → Device toolbar)

Cara Deploy Online:

GitHub Pages

Gratis, mudah, upload ke GitHub repository

Netlify

Drag & drop folder website

Firebase

Google hosting dengan CLI tools
