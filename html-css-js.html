Mengenal Teknologi Web

HTML, CSS, JavaScript & Tailwind CSS

HTML (HyperText Markup Language)

Apa itu HTML?

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar halaman web. HTML menggunakan tag-tag untuk mendefinisikan elemen seperti heading, paragraf, link, gambar, dan lainnya.

Fungsi HTML:

  • 🏗️ Membuat struktur dan kerangka website
  • 📝 Menampilkan konten seperti teks, gambar, video
  • 🔗 Membuat hyperlink antar halaman
  • 📋 Membuat form input data

Contoh Kode HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Website Saya</title>
  </head>
  <body>
    <h1>Selamat Datang!</h1>
    <p>Ini adalah website pertama saya.</p>
  </body>
</html>

CSS (Cascading Style Sheets)

Apa itu CSS?

CSS adalah bahasa styling yang digunakan untuk mengatur tampilan dan layout halaman web. CSS mengontrol warna, font, spacing, animasi, dan responsive design.

Fungsi CSS:

  • 🎨 Mengatur warna, font, dan tampilan visual
  • 📐 Mengatur layout dan posisi elemen
  • 📱 Membuat website responsive untuk mobile
  • ✨ Menambahkan animasi dan efek interaktif

Contoh Kode CSS:

.header {
  background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
  color: white;
  padding: 40px;
  text-align: center;
  border-radius: 15px;
}

JavaScript

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang membuat website menjadi interaktif dan dinamis. JavaScript dapat merespons klik user, mengubah konten secara real-time, dan berkomunikasi dengan server.

Fungsi JavaScript:

  • 🖱️ Menangani interaksi user (klik, hover, input)
  • 🔄 Mengubah konten website secara dinamis
  • 📡 Berkomunikasi dengan API dan database
  • 🎮 Membuat animasi dan game sederhana

Contoh Kode JavaScript:

function openLightbox(index) {
  this.currentIndex = index;
  this.updateImage();
  this.lightbox.classList.add('active');
  document.body.style.overflow = 'hidden';
}

Tailwind CSS

Apa itu Tailwind CSS?

Tailwind CSS adalah utility-first CSS framework yang menyediakan kelas-kelas siap pakai untuk styling. Berbeda dengan CSS tradisional, Tailwind menggunakan kelas utility seperti "bg-blue-500" atau "text-center".

Keunggulan Tailwind:

  • ⚡ Development lebih cepat dengan utility classes
  • 📦 File size lebih kecil (hanya class yang dipakai)
  • 🎯 Konsistensi design system
  • 📱 Built-in responsive design

Perbandingan:

CSS Tradisional:
.button {
  background-color: #3b82f6;
  color: white;
  padding: 12px 24px;
  border-radius: 8px;
}

Tailwind CSS:
<button class="bg-blue-500 text-white px-6 py-3 rounded-lg">
